# emu-gb

A Game Boy emulator written from scratch in C.

## Building

### Requirements

- gcc
- make __OR__ cmake

If using Nix, you can download all requirements through `nix-shell`.

### Linux / MacOs

You can build using either Make or CMake.

```sh
# Using make
make

# Using CMake
mkdir build
cd build && cmake ..
make
```
